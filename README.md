## 🚀 Como executar

Clone o projeto e acesse a pasta do mesmo.

```bash
$ git clone https://github.com/zdog10127/Flowers
$ cd Flowers
```

Para iniciá-lo, siga os passos abaixo:
```bash
# Iniciar o projeto
$ start index.html
```
O app estará disponível no seu browser pelo endereço http://localhost:5500.
